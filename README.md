# Research-on-automated-penetration-testing-based-on-bi-agent-reinforcement-learning
 项目概述

本项目提出了一种创新的"结构探索-漏洞利用"双智能体分层强化学习框架，用于自动化网络渗透测试。该框架通过任务解耦，显著提升了在复杂企业网络环境中的攻击效率与成功率。
✨ 核心特性

    双智能体分层架构：宏观结构探索与微观漏洞利用的智能分工

    基于PPO的协同训练：独立智能体通过协同奖励机制实现高效合作

    大规模仿真环境：支持 LargeNetWork-v0 和 HugeNetWork-v0 两种复杂网络环境

    完整基线对比：与DQN、PPO、SAC等主流强化学习算法的系统对比
